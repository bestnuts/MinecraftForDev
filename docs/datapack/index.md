# 설명
데이터팩은 서버 측에서 불러올 데이터 요소들을 담고 있습니다.  
또한 함수를 통해서 시스템 로직을 구축할 수 있습니다.  
데이터팩 문서 내에서 모든 팩 경로 기준은 pack.mcmeta 위치로 시작합니다.
