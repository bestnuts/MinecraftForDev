설명

이미지 파일을 문자로 가져올때 쓰입니다.

포맷

type : bitmap
ascent : 문자의 출력 높이 위치를 결정합니다.
height : 출력 크기를 결정하며 기본 값은 8 입니다.
chars : 대체할 문자를 지정합니다.
file : .png 리소스 경로

팁

ascent 는 height 보다 높을 수 없습니다.
문자의 너비는 알파 값이 0 보다 큰 오른쪽 마지막 위치를 기준으로 됩니다.
적용되는 이미지 파일은 256x256 크기를 넘길 수 없습니다.
